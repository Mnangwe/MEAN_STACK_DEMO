What is MEAN stack?

It is the use of one the popular Javascript Tech-stack that comprises of:

  • MongoDb (M) - (NoSQL Database System - Backend);
  • Express (E) - (Node Web Framework - Backend);
  • Angular (A) - (Javascript Framework - Frontend);
  • NodeJS (N) - (Javascript Runtime Environment)

The use of these technologies combined form what is know as Fullstack, which means the relationship and interaction between the Backend technologies and Frontend technologies to control the use of data and interacting with the data.

In this tutorial, we will show you how to make Angular 14 connect to MongoDB with Node.js Express. We are going to build a full-stack Angular + Node.js + MongoDB example (MEAN stack CRUD) in which, the back-end server uses Node.js + Express for REST APIs, front-end side is an Angular App with HttpClient, Router and Bootstrap.

This assumes that you have basic information of these technologies and done some research.
